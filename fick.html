Background

This is a Python package that I created in my free time. I saw that there was no open source python package for performing diffusion simulations with Fick's Second law of diffusion. This package could be used to model Solid state diffusion in the specified geometries.

Checkout the Package!

In the semiconductor industry, solid solutions of boron and silicon are used to create P-type semiconductors. The added Boron in the silicon provides extra electrons that make the doped silicon more conductive. So how does one create a silicon chip with a specific composition of Boron? Initially a layer of Boron is added to the outside of the Silicon chip. The chip is then heated to allow for the Boron to diffuse into the silicon. Based on the desired composition the sample is held at the desired temperature until the desired composition is reached. The red line symbolizes the initial thickness of the Boron layer.

Notice that the Boron permeates into the Silicon and loses it intital 100% composition.
